# FreeDom Tunnel Pro

Minimal skeleton Android app (Java) implementing:
- SSH connection using JSch
- Foreground Service
- Basic UI (Host/Port/User/Pass)
- Placeholder for SOCKS5 proxy

This is a starter project generated for you. To build:
- Import the project into Android Studio (use Gradle plugin 7.4.2+)
- Ensure internet permission and foreground service permission are allowed
- JSch library is declared in app/build.gradle
